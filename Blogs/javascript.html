Is javascript compiled or interpreted language?

Introduction

Javascript is interpreted line by line by the browser , it is done by javascript engine. javascript engine is different for diffrene browsers. javascript executes libe by line, theres no compilation like in java. but javascript falls under the general category of "dynamic" or "interpreted" languages, but it is a compiled language

compilation process

In Traditional compiled language process invloves three steps in compilation process
  1. Tokennizing
  2. parsing
  3. code generation

Tokennizing

Breaking up a string of characters into meaningfull chunks called as tokens. for example consider a line of code const pi = 3.14;. This can be broken into const , pi , = , 3.14 , ; .

Parsing

Taking a array of tokens and turning it into a tree of nested elements, which collectively represents "Abstrct Syntax tree"

  • const is a variable declaration
  • pi is a identifer name
  • = assignment Expression
  • 3.12 is a floating point literal

code generation

The Process of taking an abstract syntax tree and turning it into an executable code is known as code generation.for example it takes const pi = 3.14 and turn it into a set of machine instructions to actually create a constant variable called pi and then store a value into pi.
